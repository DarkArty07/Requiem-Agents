# Requiem Agents

Multi-agent orchestration system with vibecoding, quality gates, and gothic horror theme.

## Architecture

- **Raven** — Assistant (hermes-agent v0.17, expensive model, vibecoding)
- **Necromancer** — Orchestrator (custom, medium model, delegates to Shades)
- **Revenant** — Auditor (custom, medium model, peer of Necromancer, veto power)
- **Shades** — Executors (custom, cheap model, domain-specific tools)

## Theme

Gothic Horror — from darkness, creation.
